# Gemini CLI Project Instructions

- **Git Workflow**: After completing any significant code changes or achieving a major milestone in this project, you must automatically execute the following git sequence without waiting for explicit permission:
  1. `git add .` to stage all changes.
  2. `git commit -m "[concise description of the changes]"` to commit the work.
  3. `git pull` to fetch any remote changes before pushing.
  4. `git push origin` to push the new commit to the remote repository.
